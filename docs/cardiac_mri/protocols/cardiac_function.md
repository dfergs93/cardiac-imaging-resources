# Cardiac Function Protocol

**Indications:** Cardiomyopathy evaluation, pre/post-chemotherapy, valve disease follow-up, congenital disease, RV assessment, pre-operative evaluation.

---

## Sequence Order

| # | Sequence | View | Notes |
|---|----------|------|-------|
| 1 | **Localizers** | 3-plane | Quick survey |
| 2 | **Dark blood HASTE** | Axial stack (thorax) | Morphology survey; pericardium; pleura |
| 3 | **Cine SSFP** | 4CH | LV/RV function, mitral + tricuspid |
| 4 | **Cine SSFP** | 2CH | LV function, mitral |
| 5 | **Cine SSFP** | 3CH (LVOT) | LVOTO assessment, aortic valve |
| 6 | **Cine SSFP** | SAX stack (base → apex in 8 mm steps) | Volumetry — primary EF / EDV / ESV measurement |
| 7 | **Cine SSFP** | RVOT | Pulmonic valve, RVOT morphology |
| 8 | **Cine SSFP** | Aortic valve (en face) | Valve morphology, orifice |
| 9 | **T2 mapping** | 3 SAX (base/mid/apex) | *Optional — if edema suspected* |

---

## Key Measurements to Report

- **LV:** EDV, ESV, SV, EF, mass (indexed to BSA)
- **RV:** EDV, ESV, SV, EF (right-heart dedicated SAX or axial stack)
- **Wall motion:** Segmental (17-segment AHA model) — describe hypokinesis, akinesis, dyskinesis
- **Pericardium:** Effusion volume, thickening, calcification
- **Valves:** Qualitative assessment of regurgitation on cine

!!! tip "SAX Volumetry Tips"
    - Include basal slices that show >50% myocardium in the plane
    - Contour from first slice with clear septal muscle (not AV plane) to apex
    - RVOT can be tricky — sometimes exclude if partially through tricuspid plane

---

## Reporting Template Structure

```
TECHNIQUE: CMR performed on [X]T scanner with cardiac gating. 
No gadolinium contrast administered.

LV: EDVi XX mL/m², ESVi XX mL/m², EF XX%, mass index XX g/m².
Segmental wall motion: [normal / regional hypokinesis — describe location]

RV: EDVi XX mL/m², EF XX%. Normal/Dilated/Reduced.

Valves: [qualitative from cine]
Pericardium: [normal / effusion / thickening]
Extracardiac: [lungs, mediastinum]
```
