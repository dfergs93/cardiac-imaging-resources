    Cardiac MRI Protocols

    Overview of standard cardiac MRI protocols and the sequences used in each.

    Protocol Primary Indication Key Sequences
    Cardiac Function Ventricular volumes, EF, wall motion Cine SSFP (SAX + LAX), dark blood morphology
    Viability Cardiomyopathy, ischemic scar, fibrosis Function + LGE + T1 mapping (ECV)
    Stress Perfusion Suspected CAD, ischemia evaluation Rest + stress perfusion, LGE
    Congenital / 4D Flow CHD, shunt quantification, flow analysis Dark blood morphology, cine, 4D flow or 2D PC
    Cardiac Masses Cardiac tumor, thrombus, pericardial mass Multi-echo T1, T2, perfusion, LGE

    Contrast Doses

    Use Dose Agent
    Standard LGE / perfusion 0.1 mmol/kg (single) Gadolinium-based (macrocyclic preferred)
    Perfusion + LGE same study 0.1 mmol/kg for perfusion; additional 0.05–0.1 for LGE if needed —
    T1 mapping (ECV) 0.1 mmol/kg Record pre-contrast T1 first

    NSF Risk

    Use macrocyclic agents (Group II) for patients with eGFR <30 mL/min/1.73m². Avoid linear agents. Discuss risk/benefit if eGFR <15 or on dialysis.
